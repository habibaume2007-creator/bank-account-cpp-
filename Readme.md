# Bank Account Management System

This project simulates a **bank account system** using **C++ classes and objects**.  
Users can **create an account, deposit money, withdraw money, and check their balance**.  

---

## Features

- Create a bank account with a unique account holder name  
- Deposit a specific amount  
- Withdraw a specific amount  
- Display current account balance  
- User-friendly menu to select actions  

---

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/habibaume2007-creator/bank-account-management.git

2. Open the project in any C++ IDE (Code::Blocks, Dev-C++, Visual Studio)

3. Compile and run main.cpp

4. Follow the on-screen menu to deposit, withdraw, or check balance
